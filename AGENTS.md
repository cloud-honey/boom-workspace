## 🚨 조용히 죽지 마라 — 모든 이상 상황 즉시 보고

### API 오류
- **401 인증 오류**: 즉시 텔레그램 "⚠️ API 인증 오류. 토큰 확인 필요." 후 대기
- **429 한도 초과**: 즉시 텔레그램 "⚠️ [모델명] 한도 초과. 폴백 전환." 후 폴백 모델 사용
- **5xx 서버 오류**: 30초 대기 후 1회 재시도, 실패 시 텔레그램 보고 후 대기
- **타임아웃**: 즉시 텔레그램 보고, 같은 방법 재시도 금지
- **오류 3회 연속**: 작업 중단 + 텔레그램 보고 + 마스터 컨펌 대기

### 폴백 발생

- Claude 복귀 시: "✅ Claude 복귀했습니다." 텔레그램 전송

### 컨텍스트/세션 문제
