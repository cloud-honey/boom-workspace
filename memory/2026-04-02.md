## 11:13 KST [haiku] 웹 검색 + 대화 맥락 + '모르면 모른다' 규칙 강화 (2차)

### 지시
- 웹 검색 기능 활성화
- 모르는 건 명확하게 "모르겠습니다"로 답변
- 대화 맥락 유지 능력 확인

### 작업

1. **웹 검색 강화** (`server_v2.py`)
   - 기존 DuckDuckGo 검색은 이미 구현됨
   - 검색 트리거 키워드 확장
     * 의문사 자동 감지: 누구, 언제, 어디, 무엇, 어떤, 왜, 몇
     * 정보성 질문: 정보, 정의, 뜻, 의미, 요청, 취업, 채용, 공고
   - 자동으로 관련 질문에 웹 검색 실행

2. **'모르면 모른다' 규칙 강화**
   - 시스템 프롬프트에 명확한 가이드라인 추가
   - 사용처 명시:
     * 사실 확인 필요 + 검색 결과 없음
     * 미래 예측 ('내일 비트코인' 등) → 절대 추측 금지
     * 개인 의견 요청 ('뭐가 좋아?') → '모르겠습니다'
     * 최신 정보 (훈련 데이터 부족)
   - 예시로 명확히: 
     * 나쁜 답: "모르겠습니다." (그냥 회피)
     * 좋은 답: "어제 소식을 찾을 수 없습니다." (명확한 이유)

3. **대화 맥락 메모리** (`booml-bot-v2.py`)
   - 사용자별 대화 히스토리 유지
     * 최근 10개 메시지 (user + assistant)
     * deque로 자동 오래된 항목 삭제
   - 메시지 전송 시 히스토리 포함
     * query_mlx() 함수에 user_id 파라미터 추가
     * 대화 흐름 자연스럽게 이어짐
   - /start로 히스토리 초기화 가능
   - 메시지 형식: {"role": "user"/"assistant", "content": "..."}
