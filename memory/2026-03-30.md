## 09:12~09:58 KST [MAC/gpt-5.4] XPS 노드 끊김 문제 해결
- 지시: XPS 노드 끊김 알림이 계속 오는 문제 확인 요청
- 원인 분석:
  - `openclaw-node` 프로세스는 실행 중이었으나 systemd 서비스 미등록 상태
  - 이전에 `/etc/systemd/system/openclaw-node.service` (root 단위)로 잘못 설치됨
  - 올바른 위치는 user 단위: `~/.config/systemd/user/openclaw-node.service`
  - 수동 실행된 프로세스라 재부팅/크래시 시 자동 재시작 안 됨
- 해결:
  1. 잘못된 systemd 서비스 제거: `sudo systemctl disable --now openclaw-node && sudo rm /etc/systemd/system/openclaw-node.service`
  2. `openclaw node install --force` 로 올바르게 재설치 (user unit)
  3. `openclaw node restart` 후 연결 확인
- 결과: ✅ XPS 노드 `paired · connected` 확인 / 자동 시작 설정 완료
- 서비스 파일: `/home/sykim/.config/systemd/user/openclaw-node.service`
- 실행 커맨드: `node /home/sykim/.npm-global/lib/node_modules/openclaw/dist/index.js node run --host kim-macmini.tail89d37c.ts.net --port 443 --tls`
- 핵심 교훈: `openclaw node install` 은 반드시 user 단위로 설치됨. sudo로 /etc/systemd/system에 수동 설치 금지
